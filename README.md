# 个人成长助手小程序介绍
## 🎯 产品定位
「个人成长助手」是一款轻量化、高自由度的个人成长管理工具，专为想要持续自我提升、掌控生活节奏的用户打造，用极简的设计帮你记录成长、复盘生活、推进目标，让每一步努力都有迹可循。

---
## ✨ 核心功能模块
### 1. 「我的存款」：可视化财富管理
- 直观展示个人存款余额，支持**存入/取出**操作，轻松记录收支、追踪财富变化
- 清晰的数字呈现，帮你养成理性消费、定期储蓄的好习惯，实现财富稳步增长

### 2. 「自我反省」：每日成长复盘
- 多维度自我评分体系，覆盖**家务、赚钱、松弛感、小确幸**等生活核心维度
- 星级评分直观展示每日状态，帮你快速复盘生活、发现不足，针对性提升自我
- 可自定义评分维度，适配不同用户的成长需求，打造专属复盘模板

### 3. 「亟待解决的问题」：目标推进管理
- 待办事项清单，清晰记录需要推进的任务与目标
- 支持添加子任务，拆解复杂目标，让大问题变成可执行的小步骤
- 任务完成勾选、删除功能，高效管理待办，告别拖延，稳步推进目标

---
## 💡 产品亮点
1.  **极简设计，零学习成本**：界面清爽无冗余，打开即用，无需复杂设置，专注核心功能
2.  **全场景适配，灵活自定义**：无论是财富管理、生活复盘还是目标推进，都能满足需求，适配学生、职场人等各类用户
3.  **轻量化工具，不占内存**：小程序形态，无需下载安装，随时打开使用，不占用手机存储空间
4.  **陪伴式成长，持续精进**：用记录和复盘的方式，帮你养成自律习惯，在日常中实现持续成长

---
## 🎯 适用人群
- 想要养成储蓄习惯、管理个人财务的用户
- 希望每日复盘、持续自我提升的成长型人群
- 被待办事项困扰、需要拆解目标、告别拖延的用户
- 追求极简生活、用工具优化日常的效率达人

---
## 📌 使用场景
- 每日睡前：用「自我反省」复盘当日状态，记录生活感悟
- 发薪日：在「我的存款」中记录收入，规划储蓄与消费
- 工作/学习中：用「亟待解决的问题」拆解任务，推进目标落地
- 月度总结：回顾存款变化、评分趋势，总结成长，规划下一阶段目标
